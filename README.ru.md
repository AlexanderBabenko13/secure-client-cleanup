# Secure Client Cleanup Utility

[English version](README.md)

Secure Client Cleanup Utility — это WPF-утилита на Windows PowerShell 5.1 для инженеров поддержки и системных администраторов, которым нужно безопасно подготовить Windows-рабочую станцию к переустановке Cisco AnyConnect / Cisco Secure Client.

## Назначение

Используйте утилиту, если Cisco AnyConnect или Cisco Secure Client работает некорректно, был удалён частично либо не устанавливается повторно без ошибок.

Цель проекта — контролируемая очистка перед переустановкой. Утилита работает с известными компонентами Cisco AnyConnect и Cisco Secure Client и не предназначена для массового удаления всего программного обеспечения или всех данных Cisco.

## Предупреждение администратора

Запускайте утилиту только с правами администратора.

Очистка может останавливать службы и процессы, удалять целевые папки и целевые ключи реестра. Перед действиями, изменяющими систему, обязательно используйте сканирование, диагностику и пробный запуск / WhatIf.

## Безопасный порядок работы

1. Сканирование
2. Пробный запуск / WhatIf
3. Резервная копия
4. Очистка
5. Перезагрузка
6. Переустановка Cisco Secure Client

## Возможности

- Сканирование служб, процессов, папок, ключей реестра, пользовательских каталогов AppData, каталогов ProgramData, установленных программ и защищённых корневых путей Cisco AnyConnect / Cisco Secure Client.
- Умный поиск в `Program Files\Cisco` и `Program Files (x86)\Cisco`.
- Безопасный пробный запуск / WhatIf без удаления данных.
- Резервное копирование реестра и папок перед удалением, если резервное копирование включено.
- Экспорт результатов в HTML, CSV и JSON.
- HTML-отчёт с экранированием содержимого.
- VPN-диагностика: DNS, ping, TCP 443, настройки прокси, маршруты, адаптеры и конфликтующие VPN-инструменты.
- Защитные ограничения, блокирующие автоматическое удаление широких корневых ключей реестра и папок Cisco.
- Сборка GUI-приложения в EXE с помощью ps2exe.

## Требования

- Windows 10 или Windows 11
- Windows PowerShell 5.1
- Права администратора для очистки и полной диагностики

## Быстрый запуск скрипта

Из корня репозитория:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\src\SecureClientCleanup.ps1
```

При необходимости bootstrap автоматически перезапускает утилиту с повышенными правами и в STA-режиме.

## Сборка EXE

Установите ps2exe:

```powershell
Install-Module ps2exe -Scope CurrentUser
```

Запустите сборку:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\build\Build-SecureClientCleanup.ps1
```

Путь по умолчанию:

```text
dist\SecureClientCleanup.exe
```

После успешной сборки скрипт выводит SHA256-хеш и информацию о созданном файле.

Дополнительные параметры описаны в [руководстве по сборке](docs/BUILD.md).

## Журналы и отчёты

Журналы, результаты сканирования, резервные копии и HTML-отчёты сохраняются в каталоге вывода, выбранном в интерфейсе.

Эти материалы могут содержать локальные пути, имена компьютеров, названия служб и адаптеров, а также диагностические сведения. Проверяйте и очищайте их перед передачей третьим лицам.

## Важные ограничения

- Широкие корневые папки Cisco не удаляются автоматически.
- `HKLM:\SOFTWARE\Cisco` и `HKCU:\Software\Cisco` заблокированы для автоматического удаления.
- `C:\Program Files\Cisco`, `C:\Program Files (x86)\Cisco` и `C:\ProgramData\Cisco` заблокированы для автоматического удаления.
- Автоматический сброс сетевых настроек не выполняется.
- `Win32_Product` не используется и не должен добавляться.
- Утилита предназначена для очистки Cisco AnyConnect / Cisco Secure Client, а не для общего удаления программного обеспечения Cisco.
- Перед очисткой обязательно просматривайте результаты пробного запуска / WhatIf.
